组件说明
<checkbox-list></checkbox-list>

属性 说明 类型 必填 不传时默认值 自动检测父页面值变化 支持.sync
v-model 组件值 数组 否，不传时只能过用getCheckedValue方法获取值 [] 是 否
:list-data 选项数据，格式为：[ {'value': '1', 'text': 'text1'}, {'value': '2', 'text': 'text2'}, {'value': '3', 'text': 'text3'} ]的数组，会按数组元素顺序显示 数组 否 undefined 是 否
list-url 获取选项数据的url，返回值按result规范，data中包含数据，格式同list-data 字符串 否 undefined 否 否
value-prop 当数据源为list-data或list-url时，指定对象的某个属性做为value 字符串 否 'value' 是 否
text-prop 当数据源为list-data或list-url时，指定对象的某个属性做为text 字符串 否 'text' 是 否
:object-data 选项数据，格式为：{'1': 'text1', '2': 'text2', '3': 'text3'}的对象，会按key顺序显示 对象 否 undefined 是 否
object-url 获取选项数据的url，返回值按result规范，data中包含数据，格式同object-data 字符串 否 undefined 否 否
选项数据优先级为：list-data > list-url > object-data > object-url
:vertical 是否纵向排列，true=纵向，false=横向 布尔值 否 false 是 否
:check-all 是否显示全选框 布尔值 否 false 是 否
:disabled 是否禁止选择 布尔值 否 false 是 否
事件 说明 参数
@checked-change 选择项发生变化时回调方法 value：选中项值，数组
方法 说明 参数 返回值
getCheckedValue() 获取选中项值 无 字符串数组
getCheckedText() 获取选中项的文本 无 字符串数组
