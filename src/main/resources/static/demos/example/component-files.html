附件下载组件
附件下载 
                <files :files="files"></files>
附件下载（无数据状态） 
                <files :files="noneFile"></files>
附件下载（文件ID赋值） 
                <files :files-id="fileIds"></files>
组件说明
属性 说明 类型 必填 不传时默认值 自动检测父页面值变化
:file JSON格式： [{ "id" : "id", "fullname" : "文件名称", "contentLength" : "文件大小", "contentType" : "文件类型" }] 数组 否 空数组 是
:files-id ['0','1'] 数组 否和files二选一 空数组 是
