  • 金额转大写
说明
转千分位

value1={{popInputData}}
value2={{popInputData2}}
组件说明
1、vue中is的作用和用法详见https://segmentfault.com/a/1190000017262264
2、popInputData中的 label表示金额文本标题；money 小写数字金额；upperMoney会根据小写数字赋值 
代码示例：
转千分位：<p is="to-thousands" :modeldata="dataList.estimatedAmount"  class="detail_line"></p> "加detail_line有下划线，用于表格"
转大写金额：<tr is="to-uppercase" :popinputdataleve1="popInputData" :popData.sync="popInputData"></tr>
数据源：
require(['jquery','vue','global'], function ($, Vue) {
        var app = new Vue({
            el: '#app',
            data: {
                popInputData:{
                    label:'金额1',
                    money:'',
                    upperMoney:''
                },
                popInputData2:{
                    label:'金额2',
                    money:'21',
                    upperMoney:''
                },
                dataList:{
                    estimatedAmount:123344555
                }
            },
    });
